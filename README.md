Cup is a lightweight menu bar tools for macos. It aims to hide and show various icons from your menu bar, which can bring you a clean environment. 

## Install 
Download .zip file from release page, then move the unzipped file to your `application` folder.

## Usage
After you running the application, there will be two icons in menu bar. You need to press `CMD` + drag the line icon to the place you want.

## Requirements
Macos Version >= 15
